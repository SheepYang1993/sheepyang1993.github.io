AndroidStudio找不到源码解决方法

  • Android
  • 源码

使用AndroidStudio2.1版本，想要查看SDK源码时，源码方法里显示

throw new RuntimeException("Stub!");

IDE上方提示 

Decompiled .class file, bytecode version XX(java X)
Sources for 'Android API XX Platform' not found.

解决方法

首先下载源码，接着打开 C:\Users\username.AndroidStudioXXX\config\options\jdk.table.xm

在 jdk.table.xm 文件中找到提示信息中对应的Android API 版本，修改源码存放路径，重启AndroidStudio就能看到源码了。
